# 项目运行说明

## 前置要求

1. **安装 Node.js**
   - 访问 https://nodejs.org/ 下载并安装 Node.js（推荐 LTS 版本）
   - 安装完成后，重启终端/命令行窗口
   - 验证安装：在命令行运行 `node --version` 和 `npm --version`

## 运行步骤

1. **安装项目依赖**
   ```bash
   npm install
   ```

2. **配置环境变量（可选）**
   - 如果项目需要使用 Gemini API，需要创建 `.env.local` 文件
   - 添加：`GEMINI_API_KEY=你的API密钥`

3. **启动开发服务器**
   ```bash
   npm run dev
   ```

4. **访问应用**
   - 开发服务器启动后，通常在浏览器访问：http://localhost:3000
   - 具体端口号请查看终端输出

## 其他命令

- **构建生产版本**：`npm run build`
- **预览生产版本**：`npm run preview`

## 已修复的问题

✅ **TrainingView.tsx** - 修复了训练循环中 epoch 状态更新的 bug

## 注意事项

- 确保 Node.js 版本 >= 16（推荐使用 18+）
- 如果遇到端口占用，可以在 `vite.config.ts` 中修改端口号
